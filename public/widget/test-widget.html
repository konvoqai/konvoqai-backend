Konvoq Chat Widget - Customization Test

Widget Configuration

This page demonstrates all available customization options for the Konvoq Chat Widget.

Attribute Value Description
api-url http://localhost:3053/api/v1/webhook Webhook endpoint URL
widget-key your-widget-key-here Your unique widget key
primary-text Custom welcome message Initial greeting message (supports markdown)
banner-text Merlin Chat Header title text
banner-text-color #1B73CC Header title text color
banner-color #CDCFD1 Header background color
bot-color #ff6699 Bot theme color (legacy, use banner-color instead)
send-color #B82E12 Send button background color
floating-btn #B82E12 Floating button background color
user-chat-color #CDCFD1 User message bubble background color
close-button-color #F527EE Close button icon color
chat-voice-icon-color #7908FB Voice icon color (for future use)
voice-send-button #7908FB Voice send button color (for future use)
logo-icon URL to logo image Custom logo URL (displays in header)
auto-open true/false Auto-open chat after 5 seconds
