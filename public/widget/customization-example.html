üîë How to Get Your Widget Key

1. Call POST /api/auth/widget/create with your configuration
2. You'll receive a unique widget key like wk_abc123...
3. Use the single-line script with your widget key
4. Update your configuration anytime with PUT /api/auth/widget/update

API Request Example

Use this to create/update your widget configuration:

// Create widget const response = await fetch('http://localhost:3053/api/auth/widget/create', { method: 'POST', headers: { 'Authorization': 'Bearer YOUR_JWT_TOKEN', 'Content-Type': 'application/json' }, body: JSON.stringify({ widgetName: 'My Chat Widget', widgetConfig: { primaryText: '**Hey there!** Ask me anything...', bannerText: 'Chat Support', bannerColor: '#CDCFD1', sendColor: '#B82E12', // ... other config options } }) }); const data = await response.json(); console.log('Embed Code:', data.data.embedCode);
